Technologies

We created the HostelAPP leveraging several technologies. In its core, HostelApp is a multi-tier Spring Boot application. Its frontend is s Spring MVC application that implements the authentication/authorization features is responsible for the look and few of the application, by using w3c standards such as HTML, CSS and JavaScript. Somoe other frameworks is also used in the front-end, as for example, BootStrap

In this page, we present details of the technologies used during development of HostelApp.
